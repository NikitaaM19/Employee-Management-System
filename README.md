# Employee-Management-System
Employee Management System for Jayant Textile is a web-based application built using ASP.NET, C# and MySQL. It automates employee records, attendance, salary, leave, loan and issue management with separate admin and employee modules. The system reduces paperwork, improves data accuracy and enhances overall workforce management efficiency.

# Employee Management System for Jayant Textile
It provides:
- Easy access for both **Admin** and **Employees**
- Centralized storage of employee data
- Secure login-based access
- Payroll system
- Attendance and leave management
- Task & communication modules

## Objectives
- Automate manual employee record handling
- Reduce paperwork and human errors
- Provide fast and secure access to employee data
- Improve productivity and management efficiency
- Provide separate dashboards for Admin and Employees

### 2. Employee
